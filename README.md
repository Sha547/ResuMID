# Resume Roaster

**Think your resume is perfect? Let AI humble you.**

This is a web application that takes your resume and uses the Gemini API to ruthlessly roast every typo, clich√©, and career gap it can find. It's not just feedback; it's a reality check.

###  Features
* **Drag & Drop Upload:** Easily upload your PDF resume.
* **AI-Powered Roasts:** utilizes Google's Gemini API to analyze and mock your professional history.
* **Intensity Slider:** You control how much it hurts.

###  Tech Stack
* **Frontend:** React, TypeScript, Vite
* **Styling:** Tailwind CSS
* **AI:** Google Gemini API
